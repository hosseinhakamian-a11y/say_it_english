## Packages
framer-motion | For smooth page transitions and animations
recharts | For visualizing student progress and test results
date-fns | For formatting dates in bookings and schedules
lucide-react | For beautiful icons
clsx | Utility for conditional class names
tailwind-merge | Utility for merging tailwind classes

## Notes
The application requires RTL direction for Persian text.
Authentication is handled via the /api/auth endpoints.
API integration uses TanStack Query.
Styling uses Tailwind CSS with a custom theme.
